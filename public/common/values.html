<p>Strings, which can be added together with <code>str</code>.</p> 
(println (str &quot;clojure&quot; &quot;lang&quot;))
 
;; &gt;&gt; clojurelang
=&gt; nil
<p>Integers and floats.</p> 
(println &quot;1+1 =&quot; (+ 1 1))
 
;; &gt;&gt; 1+1 = 2
=&gt; nil
 
(println &quot;7.0/3.0 =&quot; (/ 7.0 3.0))
 
;; &gt;&gt; 7.0/3.0 = 2.3333333333333335
=&gt; nil
<p>Booleans, with boolean operators as you'd expect.</p> 
(println (and true false))
 
;; &gt;&gt; false
=&gt; nil
 
(println (or true false))
 
;; &gt;&gt; true
=&gt; nil
 
(println (not true))
 
;; &gt;&gt; false
=&gt; nil
